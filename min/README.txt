The files in this directory can be used to include humlib
in an external project without needing to copy the entire
repository.

* humlib.h   = Combined include file for humlib.
* humlib.cpp = Combined src file for humlib.

XML parser (from https://pugixml.org) used in humlib.

* pugixml.hpp    = Header file for pugixml library.
* pugiconfig.hpp = Configuration file for pugixml library.
* pugi.cpp       = Source code for pugixml library.

Standard MIDI file library (needed for tools that have MIDI file input/output).

* midifile.h   = Header file for midifile library.
* midifile.cpp = Source code for midifile library.
